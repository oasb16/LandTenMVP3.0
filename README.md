# LandTenMVP3.0

Monorepo for a modern SaaS platform using a managed mix architecture.

## Structure
- `frontend/` - Next.js/React app
- `backend/` - FastAPI app
- `functions/` - Serverless functions (AI, chat, etc.)
- `infra/` - Infrastructure as code
- `scripts/` - Utility scripts
- `docs/` - Documentation
